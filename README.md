# Add It Up

## Overview
Add It Up is a small expense‑tracking app created to help people keep better control of their daily spending. I designed this project after seeing how hard it can be to manage expenses consistently, especially when money is tight or unexpected costs come up. 
The goal is to make tracking simple: enter an amount, choose a category, and review your spending in one place.

## Features
- Add expenses with amount, category, and date

- View all saved expenses

- View total spending

- Track spending by category

- Set a monthly budget

- Compare spending to budget

- Save and load data for persistence
